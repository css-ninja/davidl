  • Idée, Papier, Crayon

    Je fais parti de ces designers incapables de commencer à faire quoi que ce soit sur un logiciel avant d’avoir posé mes idées sur papier. Avant je jetais ces maquettes directement à la poubelle, mais ça c’était avant. Maintenant je prends des photos pour conserver mes idées et les partager sur ce site.

  • Guide CSS

    Pendant des années j’ai travaillé sur des projets où le CSS peut être comparé à un bidonville : Un espace où s’entassent divers éléments, dans un chaos le plus complet. J’ai rédigé ce guide qui regroupe les bonnes pratiques pour coder des feuilles de styles de qualité, évolutives et maintenables.

  • Pokemon breakpoint

    Les développeurs front-end ont un problème dans la conception adaptative. Nous fixons des points d’arrêt en CSS et peut-être que nous voulons les utiliser dans notre JavaScript. Pokemon breakpoint est une façon amusante de faire le lien entre ces 2 mondes.
