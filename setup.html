Instructions d’installation

Les instructions suivantes vous aiderons à être prêt pour suivre la formation git & GitHub dans de bonnes conditions.

  • Choisir un bon éditeur de texte
  • Installer un bon gestionnaire de paquet
  • Installer git et associer votre profil GitHub
  • Personnaliser votre Terminal

Sublime Text 3 - Votre éditeur

Un bon editeur de texte est l’outil le plus important pour un developpeur web.
Tapez ces commandes dans un Terminal:

  sudo add-apt-repository ppa:webupd8team/sublime-text-3
  sudo apt-get update
  sudo apt-get install -y sublime-text-installer

Installer le gestionnaire des paquets

Pour télécharger automatiquement les mises à jour de votre éditeur de texte, vous devez installer un gestionnaire de paquets. Vous pouvez installer le nombre de paquet que vous souhaitez, mais commencez par les 5 paquets indispensables est une bonne base.

Terminez votre installation en hackant l’interface de Sublime Text.

Git & GitHub

Téléchargez git, le logiciel de gestion de versions qui vous permettra d’archiver votre code :

  sudo add-apt-repository ppa:git-core/ppa
  sudo apt-get update
  sudo apt-get install -y git

Dès que l’installation est terminée, vous devez configurer git & GitHub.

Terminal

Nous allons utiliser un shell nommé zsh au lieu de bash, celui par default.

  sudo apt-get install zsh

Ensuite vous devez installer oh-my-zsh pour bénéficier de pleins de raccourcis trop cools !

  sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

Pour afficher les changements, vous dever redemarrer votre machine :

  sudo shutdown -r 0
